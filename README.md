# Crystal

Crystal 是一个在终端运行的智能编程助手，理解你的代码库并通过自然语言指令执行常规任务。

## 为什么是 Crystal

- **面向真实代码库**：围绕“在仓库里定位、修改、验证”的工作流设计，不是只生成片段代码。
- **内置工具**：读文件 / 搜索 / 写文件 / 执行命令，覆盖最常见的开发动作。
- **默认更安全**：写文件与执行命令默认关闭，按需显式开启，避免误操作。
- **可复盘**：每次会话会落盘 JSONL 日志，便于回放与定位问题。
- **交互 + 脚本化**：既支持交互式 `chat`，也支持一次性 `run` 方便自动化。

## 项目运行

```bash
bun install
bun link
```

安装完成后：

```bash
crystal --help
```
